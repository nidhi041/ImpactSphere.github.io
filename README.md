# 🛠️ Hackathon Volunteer Management System

> Built with 💡 by **Team Impact Sphere** during Hackathon - Innov8
> A powerful platform to streamline volunteer management, registration, and task coordination for hackathons.

---

## 🚀 Project Overview

The **Hackathon Volunteer Management System** is a full-stack web application designed to help organizers efficiently manage volunteers during large-scale events. It offers tools for registration, role assignment, task tracking, and real-time updates.

This project was built as part of a hackathon initiative by our team to make event management easier, smarter, and more connected.

## 🧠 Key Features

- ✅ Volunteer Registration with Authentication
- ✅ Role Assignment (Admin, Team Lead, Volunteer)
- ✅ Task Tracking Dashboard
- ✅ Realtime Notifications (Socket.io)
- ✅ Wishlist / Task Preference Feature
- ✅ Admin Panel for Approvals & Communication
- ✅ Mobile Responsive UI with Dark Theme + Animations
